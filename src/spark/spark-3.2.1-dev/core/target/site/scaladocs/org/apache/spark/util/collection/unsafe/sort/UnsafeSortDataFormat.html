org.apache.spark.util.collection.unsafe.sort

UnsafeSortDataFormat 

final class UnsafeSortDataFormat extends SortDataFormat[RecordPointerAndKeyPrefix, LongArray]

Supports sorting an array of (record pointer, key prefix) pairs. Used in UnsafeInMemorySorter.

Within each long[] buffer, position 2 * i holds a pointer to the record at index i, while position 2 * i + 1 in the array holds an 8-byte key prefix.
