CancellableFSDataOutputStream 

abstract class CancellableFSDataOutputStream extends FSDataOutputStream

An interface to add the cancel() operation to FSDataOutputStream. This is used mainly by CheckpointFileManager.createAtomic to write a file atomically.

See also

CheckpointFileManager.
