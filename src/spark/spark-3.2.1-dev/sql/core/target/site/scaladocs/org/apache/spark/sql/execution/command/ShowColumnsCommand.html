ShowColumnsCommand 

case class ShowColumnsCommand(databaseName: Option[String], tableName: TableIdentifier, output: Seq[Attribute]) extends LogicalPlan with LeafRunnableCommand with Product with Serializable

A command to list the column names for a table.

The syntax of using this command in SQL is:

SHOW COLUMNS (FROM | IN) table_identifier [(FROM | IN) database];
